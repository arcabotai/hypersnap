Open-source node operator toolkit

Run Hypersnap without screenshot archaeology.

One-line install. One command to diagnose. One sanitized report when you need help.

curl -fsSL https://hypersnap.org/install.sh | bash
hypersnap doctor
GitHub Commands

Install

Bootstrap Docker, clone the operator toolkit, and point users at the upstream Hypersnap flow.

hypersnap install

Doctor

Checks Docker, Compose, containers, ports, disk, memory, info endpoint, and recent logs.

hypersnap doctor --json

Share

Creates a sanitized diagnostic bundle for Farcaster, GitHub issues, or Discord support.

hypersnap share

No surprise chainsaw.

hypersnap doctor --fix is conservative by design. It should never silently wipe chain data, expose private dashboards, change keys, or mutate dangerous infra. Risky repairs get explicit commands and warnings.

For support

Instead of “it is broken” plus a blurry screenshot, ask operators for:

hypersnap share

The report is local-first and redacts obvious secrets. Review before posting publicly. Computers are helpful. They are not priests.
